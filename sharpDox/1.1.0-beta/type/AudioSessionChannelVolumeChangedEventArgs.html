Properties

ChangedChannel

Gets the index of the audio channel that changed. Use this value as an index into the ChannelVolumes. If the session submix contains n channels, the channels are numbered from 0 to n– 1. If more than one channel might have changed, the value of ChangedChannel is (DWORD)(–1).

public Int32 ChangedChannel { get; set; }

ChannelCount

Gets the number of audio channels in the session submix.

public Int32 ChannelCount { get; set; }

ChannelVolumes

Gets the volume level for each audio channel. Each volume level is a value in the range 0.0 to 1.0, where 0.0 is silence and 1.0 is full volume.

public Single[] ChannelVolumes { get; set; }

Item
