Remarks

Context value for the OnNotify method. This member is the value of the event-context GUID that was provided as an input parameter to the AudioEndpointVolume method call that changed the endpoint volume level or muting state. For more information, see .

MasterVolume

Specifies the current master volume level of the audio stream. The volume level is normalized to the range from 0.0 to 1.0, where 0.0 is the minimum volume level and 1.0 is the maximum level. Within this range, the relationship of the normalized volume level to the attenuation of signal amplitude is described by a nonlinear, audio-tapered curve.

public  Single MasterVolume

Muted

A value indicating whether the audio stream is currently muted. true if the audio stream is currently muted; otherwise, false.

public  NativeBool Muted
