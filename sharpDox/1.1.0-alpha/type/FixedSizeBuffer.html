public class FixedSizeBuffer<T> : IDisposable where T : object

Summary

Represents a read- and writeable buffer which can hold a specified number of elements.

Type Parameters

T

Specifies the type of the elements to store.

Read(T[] buffer, Int32 offset, Int32 count)

Reads a sequence of elements from the internal buffer of the FixedSizeBuffer.

public Int32 Read(T[] buffer, Int32 offset, Int32 count)

Parameters

buffer

T[]

An array of elements. When this method returns, the buffer contains the specified array with the values between offset and (offset + count - 1) replaced by the elements read from the internal buffer.

offset

Int32

The zero-based offset in the buffer at which to begin storing the data read from the internal buffer.

count

Int32

The maximum number of elements to read from the internal buffer.

Returns

The total number of elements read into the buffer.
