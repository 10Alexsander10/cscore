Returns a new instance of the Equalizer class with 10 preset EqualizerFilter.

public static Equalizer Create10BandEqualizer(ISampleSource source, Int32 bandWidth, Int32 defaultGain)

Parameters

source

ISampleSource

The underlying sample source which provides the data for the equalizer.

bandWidth

Int32

The bandwidth to use for the 10 EqualizerFilter. The default value is 18.

defaultGain

Int32

The default gain to use for the 10 EqualizerFilter. The default value is zero which means that the data, passed through the equalizer won't be affected by the EqualizerFilter.

Returns

A new instance of the Equalizer class with 10 preset EqualizerFilter.

Read(Single[] buffer, Int32 offset, Int32 count)

Reads a sequence of samples from the underlying BaseSource, applies the equalizer effect and advances the position within the stream by the number of samples read.

public Int32 Read(Single[] buffer, Int32 offset, Int32 count)

Parameters

buffer

Single[]

An array of floats. When this method returns, the buffer contains the specified float array with the values between offset and (offset + count - 1) replaced by the floats read from the current source.

offset

Int32

The zero-based offset in the buffer at which to begin storing the data read from the current stream.

count

Int32

The maximum number of samples to read from the current source.

Returns

The total number of samples read into the buffer.
