Namespace: CSCore.Streams

Syntax

public interface IFadeStrategy

Summary

Provides a mechanism for fading in/out audio.

Remarks

The SampleRate- and the Channels-property must be set before the IFadeStrategy can be used.

Used by

  • FadeInOut

StartFading(Nullable from, Single to, Double duration)

Starts fading from a specified volume to another volume.

public abstract void StartFading(Nullable from, Single to, Double duration)

Parameters

from

Nullable

The start volume in the range from 0.0 to 1.0. If no value gets specified, the default volume will be used. The default volume is typically 100% or the current volume.

to

Single

The target volume in the range from 0.0 to 1.0.

duration

Double

The duration in milliseconds.
