Summary

Methods

GetCodec(String filename)

Returns a fully initialized IWaveSource instance which is able to decode the specified file. If the specified file can not be decoded, this method throws an NotSupportedException.

public IWaveSource GetCodec(String filename)

Parameters

filename

String

Filename of the specified file.

Returns

Fully initialized IWaveSource instance which is able to decode the specified file.

Exceptions

NotSupportedException

The codec of the specified file is not supported.

GetCodec(Uri uri)

Returns a fully initialized IWaveSource instance which is able to decode the audio source behind the specified uri. If the specified audio source can not be decoded, this method throws an NotSupportedException.

Uri which points to an audio source.

GetSupportedFileExtensions()

Returns all the common file extensions of all supported codecs. Note that some of these file extensions belong to more than one codec. That means that it can be possible that some files with the file extension abc can be decoded but other a few files with the file extension abc can't be decoded.

public String[] GetSupportedFileExtensions()

Returns

Supported file extensions.

public void Register(object key, CodecFactoryEntry codec)

key

The key which gets used internally to save the codec in a Dictionary. This is typically the associated file extension. For example: the mp3 codec uses the string "mp3" as its key.
