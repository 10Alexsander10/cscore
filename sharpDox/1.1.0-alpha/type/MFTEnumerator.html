EnumerateTransforms(Guid category, MFTRegisterTypeInfo inputType, MFTRegisterTypeInfo outputType)

Enumerates Media Foundation transforms (MFTs) in the registry.

public static Guid[] EnumerateTransforms(Guid category, MFTRegisterTypeInfo inputType, MFTRegisterTypeInfo outputType)

Parameters

category

Guid

A Guid that specifies the category of MFTs to enumerate. For a list of MFT categories, see MFTCategories.

inputType

MFTRegisterTypeInfo

Specifies an input media type to match. This parameter can be null. If null, all input types are matched.

outputType

MFTRegisterTypeInfo

Specifies an output media type to match. This parameter can be null. If null, all output types are matched.

Returns

An array of CLSIDs. For more information, see .

Remarks

On Windows 7/Windows Server 2008 R2, use the EnumerateTransformsEx method instead.
