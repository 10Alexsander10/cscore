Summary

The Mp3Format class describes an MPEG Audio Layer-3 (MP3) audio format.

Fields

BlockSize

Block size in bytes. This value equals the frame length in bytes x FramesPerBlock. For MP3 audio, the frame length is calculated as follows: 144 x (bitrate / sample rate) + padding.

public  Int16 BlockSize

CodecDelay

Encoder delay in samples. If you do not know this value, set this structure member to zero.

public  Int16 CodecDelay

Flags

Indicates whether padding is used to adjust the average bitrate to the sampling rate.

public  Mp3PaddingFlags Flags

FramesPerBlock

Number of audio frames per block.

public  Int16 FramesPerBlock
