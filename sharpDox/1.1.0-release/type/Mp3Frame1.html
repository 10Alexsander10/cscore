FindNextFrame(Stream stream, Int64 streamOffset)

Initializes the Audio-Frame instance by searching for the next frame in the stream.

public Boolean FindNextFrame(Stream stream, Int64 streamOffset)

Parameters

stream

Stream

The Stream that contains audio-frames.

streamOffset

Int64

A offset, used to calculate the absolute StreamPosition. Used in combination with the FrameScanner when the original stream got split up into multiple segments. Calculate the StreamPosition with the following formula: [Position of stream] + streamOffset.

Returns

True if a frame was found. False if no frame was found.

Remarks

If this method returns True, the StreamPosition property must be set from within the FindNextFrame method.
