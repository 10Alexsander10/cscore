GetMilliseconds(IAudioSource source, Int64 elementCount)

Converts a duration in raw elements to a duration in milliseconds. For more information about "raw elements" see remarks.

public static Int64 GetMilliseconds(IAudioSource source, Int64 elementCount)

Parameters

source

IAudioSource

The IAudioSource instance which provides the WaveFormat used to convert the duration in "raw elements" to a duration in milliseconds.

elementCount

Int64

The duration in "raw elements" to convert to duration in milliseconds.

Returns

The duration in milliseconds.

Exceptions

System.ArgumentNullException

source or elementCount

Remarks

The term "raw elements" describes the elements, an audio source uses. What type of unit an implementation of the IAudioSource interface uses, depends on the implementation itself. For example, a IWaveSource uses bytes while a ISampleSource uses samples. That means that a IWaveSource provides its position, length,... in bytes while a ISampleSource provides its position, length,... in samples.

To get the length or the position of a IAudioSource as a TimeSpan value, use the GetLength or the GetPosition property.

Internally this method uses the TimeConverterFactory class.
