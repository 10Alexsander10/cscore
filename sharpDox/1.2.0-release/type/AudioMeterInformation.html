GetChannelsPeakValuesNative(Int32 channelCount, Single[]& peakValues)

Gets the peak sample values for all the channels in the audio stream.

public Int32 GetChannelsPeakValuesNative(Int32 channelCount, Single[]& peakValues)

Parameters

channelCount

Int32

The channel count. This parameter also specifies the number of elements in the peakValues array. If the specified count does not match the number of channels in the stream, the method returns error code E_INVALIDARG.

peakValues

Single[]&

An array of peak sample values. The method writes the peak values for the channels into the array. The array contains one element for each channel in the stream. The peak values are numbers in the normalized range from 0.0 to 1.0. The array gets allocated by the GetChannelsPeakValuesNative method.

Returns
