PlayCpuOverhead

The processing overhead as a percentage of main processor cycles needed to mix this sound buffer. For hardware buffers, this member will be zero because the mixing is performed by the sound device. For software buffers, this member depends on the buffer format and the speed of the system processor.

public  Int32 PlayCpuOverhead

Remarks

UnlockTransferRate

The rate, in kilobytes per second, at which data is transferred to the buffer memory when Unlock is called. High-performance applications can use this value to determine the time required for Unlock to execute. For software buffers located in system memory, the rate will be very high because no processing is required. For hardware buffers, the rate might be slower because the buffer might have to be downloaded to the sound card, which might have a limited transfer rate.

public  Int32 UnlockTransferRate
